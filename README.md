# # 💼 Personal Portfolio Website

A modern, responsive, and interactive personal portfolio website built using HTML, CSS, and JavaScript. This portfolio showcases my skills, projects, education, and contact information with a clean dark-themed user interface.


## ✨ Features

- 🎨 Modern Dark Theme UI
- 📱 Fully Responsive Design
- 👤 About Me Section
- 🛠 Skills Section
- 💼 Projects Showcase
- 📞 Contact Form
- 🌐 Social Media Links
- 🚀 Fast Loading

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Font Awesome
- Google Fonts

---


---

## 📋 Sections Included

- Home
- About
- Skills
- Projects
- Contact

---

## 🎯 Purpose

The main purpose of this portfolio is to showcase my web development skills, projects, and experience while providing an easy way for recruiters and clients to connect with me.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/hunzalaarshad/portfolio.git
```

Go to the project folder:

```bash
cd portfolio
```

Open `index.html` in your browser.
